# Pac-Man-INF-110-UFV-
Trabalho proposto pelo professor André Gustavo durante o primeiro período do curso de Ciência da Computação na Universidade Federal de Viçosa, o objetivo era aplicar todos os conceitos e lógica aprendidos durante o semestre em um só projeto robusto e divertido. Participaram deste projeto Victor Mendes, Rhuan Machado e Paulo Vitor Martins Silva
